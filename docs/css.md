# CSS

To customize the appearance of the date picker, you can use the following CSS classes.

```css
.dp-nav-wrapper {
  /* Navigation wrapper for the date picker, which contains month and year toggles and arrow buttons. */
}

.dp-nav-month-toggle {
  /* Month toggle button in the date picker navigation. */
}

.dp-nav-month-toggle:hover {
  /* Hover state for the month toggle button in the date picker navigation. */
}

.dp-nav-year-toggle {
  /* Year toggle button in the date picker navigation. */
}

.dp-nav-year-toggle:hover {
  /* Hover state for the year toggle button in the date picker navigation. */
}

.dp-nav-left-arrow {
  /* Left arrow button in the date picker navigation. */
}

.dp-nav-right-arrow {
  /* Right arrow button in the date picker navigation. */
}

.dp-weekday {
  /* Style for the weekday name in the date picker. */
}

.dp-weekday-wrapper {
  /* Wrapper element for the weekday name in the date picker. */
}

.dp-days-wrapper {
  /* Wrapper for the days in the date picker. */
}

.dp-single-month-day {
  /* Style for the day cell in a single month calendar. */
}

.dp-multi-month-day {
  /* Style for the day cell in a multi-month calendar. */
}

.dp-single-month-day:hover {
  /* Hover state for the single month day cell in the date picker. */
}

.dp-multi-month-day:hover {
  /* Hover state for the multi-month day cell in the date picker. */
}

.dp-multi-month-day-selected {
  /* Style for the selected multi-month day cell in the date picker. */
}

.dp-multi-month-day-selected:hover {
  /* Hover state for the selected multi-month day cell in the date picker. */
}

.dp-multi-month-day-start {
  /* Style for the start day cell of a range of days in the date picker. */
}

.dp-multi-month-day-end {
  /* Style for the end day cell of a range of days in the date picker. */
}

.dp-single-month-day-selected {
  /* Style for the selected single month day cell in the date picker. */
}

.dp-single-month-day-selected:hover {
  /* Hover state for the selected single month day cell in the date picker. */
}

.dp-day-highlight-today {
  /* Style for the highlight cell of todayâ€™s date in the date picker. */
}

.dp-month-select-wrapper {
  /* Wrapper for the month select wrapper in the date picker. */
}

.dp-month-select-cell {
  /* Style for a single cell in the month select in the date picker. */
}

.dp-month-select-cell:hover {
  /* Hover state for the month select cell in the date picker. */
}

.dp-month-select-active-cell {
  /* Style for the active cell in the month select in the date picker. */
}

.dp-year-select-wrapper {
  /* Wrapper for the year select wrapper in the date picker. */
}

.dp-year-select-cell {
  /* Style for a single cell in the year select in the date picker. */
}

.dp-year-select-cell:hover {
  /* Hover state for the year select cell in the date picker. */
}

.dp-year-select-active-cell {
  /* Style for the active cell in the year select in the date picker. */
}

.dp-month-wrapper {
  /* Wrapper for the month wrapper in the date picker. */
}

.dp-background {
  /* Background color of the date picker. */
}
```
